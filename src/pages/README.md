# Pages

## Overview
Application page components for the portfolio. Each page represents a distinct section of the website.

## Files
- [Components.tsx](pages/Components.tsx) - FUI HUD components showcase
- [Contact.tsx](pages/Contact.tsx) - Contact page
- [Experience.tsx](pages/Experience.tsx) - Work experience page
- [Home.tsx](pages/Home.tsx) - Home/landing page
- [index.ts](pages/index.ts) - Page exports

## Purpose
Defines the main routes and content pages of the portfolio application using React Router.

## Exports
- HomePage - Landing page component
- ExperiencePage - Professional experience page
- ContactPage - Contact information page
- ComponentsPage - Component library showcase
