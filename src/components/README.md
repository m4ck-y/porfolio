# Components

## Overview
Reusable FUI HUD UI components for the portfolio. Contains form controls, buttons, cards, and decorative elements.

## Files
- [AIModal.tsx](components/AIModal.tsx) - AI chat assistant modal
- [FuiBackgrounds.tsx](components/FuiBackgrounds.tsx) - Background textures
- [FuiBadge.tsx](components/FuiBadge.tsx) - Status badges and chips
- [FuiButton.tsx](components/FuiButton.tsx) - Button component
- [FuiCards.tsx](components/FuiCards.tsx) - Decorated card component
- [FuiControls.tsx](components/FuiControls.tsx) - Toggle, slider, progress
- [FuiCryptoTicker.tsx](components/FuiCryptoTicker.tsx) - Crypto/stock ticker
- [FuiDecorative.tsx](components/FuiDecorative.tsx) - Corner brackets, data pills
- [FuiForms.tsx](components/FuiForms.tsx) - Select, checkbox, radio, file upload
- [FuiInput.tsx](components/FuiInput.tsx) - Text input component
- [FuiInitiateButton.tsx](components/FuiInitiateButton.tsx) - Animated initiate button
- [FuiNetworkTraffic.tsx](components/FuiNetworkTraffic.tsx) - Network traffic display
- [FuiPulsingDots.tsx](components/FuiPulsingDots.tsx) - Pulsing status indicators
- [FuiTable.tsx](components/FuiTable.tsx) - Data table component
- [FuiTooltip.tsx](components/FuiTooltip.tsx) - Tooltip component
- [FuiTypography.tsx](components/FuiTypography.tsx) - Typography styles
- [index.ts](components/index.ts) - Component exports

## Purpose
Provides a comprehensive set of sci-fi themed UI components following FUI HUD design system with cyan color palette.
